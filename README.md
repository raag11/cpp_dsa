# C++ Data Structures and Algorithms

This repository contains my structured implementations of core Data Structures and Algorithms in C++. Every solution is categorized by its underlying algorithmic pattern rather than random problem names.

## Repository Architecture

* **/basics:** Core logic building, mathematical algorithms (Prime, GCD, Armstrong), and bit manipulation.
* **/loops:** Iterative structural problems and pattern printing.
* **/arrays:** Memory manipulation, Two-Pointer technique, and Sliding Window problems.

## Execution Standards

Every C++ file in this repository strictly adheres to the following rules:
1. **Raw Implementation:** Logic is built from scratch without relying on external or overly abstracted STL functions where core algorithmic understanding is required.
2. **Complexity Analysis:** Solutions are optimized for minimum Time Complexity (Big O) and Space Complexity. Brute-force approaches are actively refactored.
3. **Edge Case Handling:** Input sanitization and standard edge cases (zero, negative inputs, boundaries) are managed within the logic.

## Daily Workflow
Code is updated daily via a strict drip-feed committing process to maintain a transparent and trackable history of logic progression.
